# 📊 HR Analytics Dashboard (Power BI)

## 📌 Project Overview

The **HR Analytics Dashboard** is an interactive Power BI dashboard designed to provide meaningful insights into an organization's workforce. It helps HR professionals and management monitor key employee metrics, identify trends, and support data-driven decision-making.

The dashboard transforms raw HR data into visual reports that make workforce analysis simple, interactive, and efficient.

## 🚀 Features

- Employee Overview
- KPI Cards for key HR metrics
- Department-wise employee analysis
- Gender distribution
- Interactive charts and visualizations
- Pivot table for detailed employee information
- Dynamic filtering and cross-highlighting
- User-friendly dashboard layout


## 📈 Dashboard Visualizations

The dashboard includes:

- KPI Cards
- Donut Chart
- Column Chart
- Bar Chart
- Pivot Table
- Interactive Filters


## 🛠️ Built With

- Power Query

## 📊 Dashboard Insights

- HR Performance Indicators
